# Visión del proyecto

Aplicación de gestión interna para empresa de producción de goma y esencias.

## Objetivo
Desarrollar una aplicación para Windows orientada a trabajo en red local, con varios usuarios, control de accesos y trazabilidad completa de acciones.

## Stack objetivo
- Python
- PySide6 para la interfaz
- PostgreSQL como base de datos central en servidor
- SQLAlchemy + Alembic
- Auditoría centralizada

## Principios
- Multiusuario en red local
- Cada trabajador con usuario y contraseña
- Registro de cualquier movimiento con fecha, hora y parte donde se realizó
- Interfaz visual y práctica para operadores
- Sistema modular, preparado para crecer
